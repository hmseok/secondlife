# 렌터카 렌트가 산정 전문 가이드

## 렌트가의 7대 원가 기둥

렌터카 월 렌탈료는 단순히 "차값 나누기 개월수"가 아닙니다.
전문 렌터카 사업자가 견적을 산출할 때 반드시 챙겨야 하는 7가지 원가 구성요소입니다.

---

## 1. 감가상각비 (가장 중요 — 원가의 50~65%)

**의미:** 계약 기간 동안 차량 가치가 하락하는 비용. 렌트가의 절반 이상을 차지.

**핵심:** 차종별/브랜드별 잔존가치율이 완전히 다름!

| 구분 | 3년 후 잔존가치 | 특징 |
|------|:---:|------|
| 국산 대형 SUV (팰리세이드) | 63% | 대기수요로 잔가 높음 |
| 국산 경차 (모닝/캐스퍼) | 62% | 실수요 많아 안정적 |
| 수입 중형 (벤츠C/BMW3) | 48% | 모델체인지 영향 큼 |
| 전기차 수입 (테슬라) | 43% | 배터리 감가 반영 |
| 수입 프리미엄 (벤츠S) | 40% | 가장 급격한 감가 |

**계산 공식:**
```
월 감가비 = (취득원가 - 잔존가치) / 계약월수

잔존가치 = 출고가 × 잔존가치율(%)
취득원가 = 차량가 + 취득세 + 공채비 + 탁송 + 부대비용
```

**주의사항:**
- `취득원가`는 차량 구매가가 아닌 **총 취득비용** 기준
- 잔존가치는 출고가 기준 (매입가 X)
- 인기 차종은 잔존가치를 보수적으로 잡아도 안전
- 수입차/전기차는 반드시 보수적 산정 필요

---

## 2. 금융비용 (원가의 15~25%)

**의미:** 차량 구매에 투입된 자본의 비용

**두 가지 구성요소:**

**A. 대출이자 (Debt Cost)**
```
월 대출이자 = 대출원금 × 연이율 / 12

일반적으로 매입가의 60~80% 대출 (LTV)
금리: 캐피탈 4.0~5.5% (기간별 차등)
```

**B. 자기자본 기회비용 (Equity Opportunity Cost)**
```
월 기회비용 = 자기자본 × 투자수익률 / 12

자기자본 = 매입가 - 대출금
투자수익률: 5~6% (정기예금/투자 대안)
```

**잔액계수 (Balance Factor):**
실무에서는 원금이 시간에 따라 줄어드는 것을 반영합니다.
```
잔액계수 = (계약월수 + 1) / (2 × 계약월수)

예시) 36개월 계약: (36+1)/(2×36) = 0.514
      → 실제 월 이자 = 대출원금 × 연이율 / 12 × 0.514
```

---

## 3. 보험료 (원가의 8~15%)

**의미:** 렌터카 영업용 종합보험

**핵심:** 렌터카는 **영업용 보험**이므로 일반 보험 대비 높음!

| 차량가 구간 | 국산 연간 | 수입 연간 | 전기차 연간 |
|------|:---:|:---:|:---:|
| ~2,000만 | 150만 | - | - |
| 2,000~3,000만 | 190만 | 280만 | 220만 |
| 3,000~5,000만 | 250만 | 350만 | 300만 |
| 5,000~7,000만 | 320만 | 420만 | 400만 |
| 7,000만+ | 400만 | 550만+ | - |

**담보 구성 (일반적):**
- 대인배상 II: 무한
- 대물배상: 2억
- 자기신체사고: 1억
- 자차보험: 차량가액 전액

---

## 4. 자동차세 (원가의 1~3%)

**핵심:** 렌터카는 **영업용**으로 비영업용 대비 크게 낮음!

| 구분 | 영업용 (렌터카) | 비영업용 |
|------|:---:|:---:|
| 1,000cc 이하 | 18원/cc | 80원/cc |
| 1,001~1,600cc | 18원/cc | 140원/cc |
| 1,601cc 초과 | 19원/cc | 200원/cc |
| 전기차 | 2만원/년 | 13만원/년 |

※ 지방교육세 30% 별도 추가

**예시:** 쏘나타 2.0 (1,999cc, 영업용)
```
기본세: 1,999 × 19 = 37,981원
교육세: 37,981 × 30% = 11,394원
연간 합계: 49,375원 → 월 약 4,115원
```

비영업용이었다면: 1,999 × 200 × 1.3 = **519,740원/년**
→ 영업용은 약 **1/10 수준**

---

## 5. 정비충당금 (원가의 3~8%)

**의미:** 계약 기간 중 발생할 정비/소모품 비용의 월 적립금

| 차종 | 신차~2년 | 3~4년 | 5년+ |
|------|:---:|:---:|:---:|
| 국산 경차/소형 | 3만 | 5만 | 8만 |
| 국산 중형 | 4만 | 6.5만 | 10만 |
| 국산 대형/SUV | 5만 | 8만 | 12만 |
| 수입차 | 7만 | 12만 | 18만 |
| 전기차 | 2만 | 3.5만 | 5.5만 |
| 하이브리드 | 3.5만 | 5.5만 | 9만 |

**전기차가 저렴한 이유:**
- 엔진오일 교환 불필요
- 회생제동으로 브레이크 패드 마모 적음
- 구동 부품 수 대폭 감소

---

## 6. 관리비/간접비 (원가의 2~5%)

**의미:** 사업 운영을 위한 고정비 할당

```
월 관리비 = 취득원가 × 관리비율(%) / 12

포함 항목:
- 인건비 (차량관리 인력)
- 사무실/주차장 임대료
- IT 시스템 비용
- 마케팅비
- 법인세 할당
```

보통 취득원가의 연 2~5% 수준

---

## 7. 이익 (목표 마진)

**두 가지 방법:**

**A. 정액 마진:** 월 10~30만원 고정 추가
```
렌트가 = 총원가 + 정액마진
```

**B. 정률 마진:** 총원가의 5~15% 추가
```
렌트가 = 총원가 × (1 + 마진율)
```

**시장 포지셔닝에 따라 조절:**
- 프리미엄: 10~15%
- 평균: 5~10%
- 가격경쟁: 3~5%

---

## 종합 산정 공식

```
월 렌탈료 = A + B + C + D + E + F + G - H

A = 월 감가비     = (취득원가 - 잔존가치) / 계약월수
B = 월 금융비     = 대출이자 + 기회비용
C = 월 보험료     = 연간보험료 / 12
D = 월 자동차세   = 연간세금 / 12
E = 월 정비충당금 = 차종/연식별 기준
F = 월 관리비     = 취득원가 × 관리비율 / 12
G = 이익          = 총원가 × 마진율
H = 보증금 효과   = 보증금 × 조달금리 / 12

청구금액 = 렌탈료 × 1.1 (VAT 10%)
```

---

## 취득원가 산정 (종종 간과되는 핵심)

```
취득원가 = 차량가 + 취득세 + 공채부담 + 탁송 + 기타

예시) 쏘나타 하이브리드 3,500만원 (서울 등록)
  차량가:     35,000,000
  취득세(7%):  2,450,000
  공채(12%):   4,200,000
  공채할인(-6%): -252,000  → 실부담: 3,948,000
  탁송료:        350,000
  번호판+인지:    27,000
  대행료:        100,000
  ─────────────────────
  취득원가:   41,875,000  (차량가 대비 +20%)
```

차량가만으로 렌트가를 산정하면 취득원가의 20%를 놓치게 됩니다!

## 기준자료 테이블 구성 (Supabase)

| 테이블 | 용도 | 데이터 수 |
|--------|------|:---------:|
| `depreciation_db` | 잔존가치율 (차종별 1~5년) | 16건 |
| `insurance_rate_table` | 보험료 기준 (차종별/가액별) | 13건 |
| `maintenance_cost_table` | 정비충당금 (차종별/연식별) | 18건 |
| `vehicle_tax_table` | 자동차세 (영업용/비영업용) | 8건 |
| `finance_rate_table` | 금리 기준 (기간별/유형별) | 11건 |
| `registration_cost_table` | 등록비용 (취득세/공채/부대) | 12건 |
| `business_rules` | 시스템 기본값 | 28건 |

---

## SQL 실행 순서

```
1. Supabase SQL Editor에서 010_pricing_reference_data.sql 실행
2. 확인 쿼리:
   SELECT * FROM depreciation_db ORDER BY category;
   SELECT * FROM insurance_rate_table ORDER BY vehicle_type, value_min;
   SELECT * FROM maintenance_cost_table ORDER BY vehicle_type, age_min;
   SELECT * FROM vehicle_tax_table ORDER BY tax_type;
   SELECT * FROM finance_rate_table ORDER BY finance_type, term_months_min;
   SELECT * FROM registration_cost_table ORDER BY cost_type;
   SELECT key, value, description FROM business_rules ORDER BY key;
```
